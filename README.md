# Organizador da Semana

Um organizador semanal feito com HTML, CSS e JavaScript.  
O projeto permite adicionar tarefas por dia da semana, definir prioridades, marcar atividades como concluídas e acompanhar melhor a rotina.

## Funcionalidades

- Adicionar novas tarefas
- Escolher o dia da semana
- Definir prioridade: baixa, média ou alta
- Marcar tarefas como concluídas
- Remover tarefas
- Filtrar por todas, hoje, pendentes e concluídas
- Contador de tarefas concluídas
- Modo claro e modo escuro
- Salvamento automático no navegador com LocalStorage
- Layout responsivo para computador e celular

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- LocalStorage

## Objetivo do Projeto

Este projeto foi criado para praticar desenvolvimento front-end e organização de interfaces interativas.

A ideia principal é oferecer uma ferramenta simples e funcional para organizar a semana, acompanhar prioridades e visualizar tarefas de forma clara.

Durante o desenvolvimento, foram praticados conceitos como:

- Estruturação de páginas com HTML
- Estilização responsiva com CSS
- Manipulação do DOM com JavaScript
- Eventos de clique e teclado
- Uso de LocalStorage para salvar dados no navegador
- Alternância entre tema claro e escuro

## Como Usar

1. Digite uma tarefa no campo de texto.
2. Escolha o dia da semana.
3. Escolha a prioridade.
4. Clique em **Adicionar**.
5. Marque a tarefa como concluída quando terminar.
6. Use os filtros para visualizar tarefas específicas.
